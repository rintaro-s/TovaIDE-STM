# UART (tmp/Cube対応)

## 概要
ESP32などの外部マイコンから受け取ったデータをSTM32で処理する。

## 最短フロー
1. ペリフェラルワークベンチで `UART` を選ぶ。
2. `ピンビュー` で USART1 のTX/RXを確認。
3. CubeMXで USART を `Asynchronous` にする。
4. ボーレートを送信側と一致させる (例: 115200)。
5. ワークベンチ生成コードを `USER CODE` に貼る。
6. 受信割込みが再開されることを確認してビルド。

## 受信処理の基本
- 初回起動時: `HAL_UART_Receive_IT(&huart1, rxBuf, sizeof(rxBuf));`
- コールバック内でも同じ受信開始を再実行する。
- ヘッダ付き固定長データにすると扱いやすい。

## データフレーム例
- byte0: header
- byte1..n: payload

## つまずき対策
- 受信が1回で止まる: コールバック末尾で再受信しているか確認。
- 文字化け: ボーレート・パリティ・ストップビットを両端で一致。
- 反応なし: TX/RX配線とGND共通を確認。
