# I2C (tmp/Cube対応)

## 概要
MPU6050などのI2Cセンサから値を読み取る。

## 最短フロー
1. ペリフェラルワークベンチで `I2C` を選ぶ。
2. CubeMXで I2C1 を有効化。
3. `mpu6050.h` を `Core/Inc`、`mpu6050.c` を `Core/Src` に追加。
4. ワークベンチ生成コードを `USER CODE PV/BEGIN 2/WHILE` に反映。
5. 初期化とキャリブレーションを1回だけ実行。
6. ループで周期読み取りしてログ確認。

## 実装ポイント
- `MPU6050_Init(&hi2c1);`
- `MPU6050_Calibration(&hi2c1, &MPU6050, 3000);`
- `MPU6050_Read_All(&hi2c1, &MPU6050);`

## つまずき対策
- 値が更新されない: SCL/SDA配線とプルアップを確認。
- 値が暴れる: キャリブレーション回数を調整。
- 制御が重い: ループ内の無駄な `HAL_Delay()` を削減。
