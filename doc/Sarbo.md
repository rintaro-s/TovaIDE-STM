# Sarbo (tmp/Cube対応 / Servo)

## 概要
サーボモータをPWM 50Hzで角度制御する。

## 最短フロー
1. ペリフェラルワークベンチで `Servo` を選ぶ。
2. TimerのPWM出力(CHx)を有効化。
3. `fPWM=50Hz` になるよう `PSC` と `ARR` を設定。
4. 推奨: `ARR=999` (0.1%刻み)。
5. 角度を compare 値へ変換するコードを生成・貼り付け。
6. 書込み後、0/90/180度で動作確認。

## 角度とDutyの目安
- 0度: 2.5%
- 180度: 12.5%
- `ARR=999` のとき compare はおおむね 25..125

## 実装ポイント
- PWM開始: `HAL_TIM_PWM_Start(&htimX, TIM_CHANNEL_Y);`
- 角度変換: `compare = min + ((max - min) * angle) / 180`
- 角度範囲は `0..180` にクランプする。

## つまずき対策
- 震える/暴れる: 電源容量不足の可能性を確認。
- 角度が合わない: min/max compareを実機に合わせて調整。
- 動かない: Timerチャンネルとピン割当を再確認。
